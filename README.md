# Snake - Python

A simple and clean implementation of the classic Snake game, built with Python.

## Prerequisites

- Python 3.x
- *(Optional, remove if you only used the standard library)* `pygame`: `pip install pygame`

## How to run

To start the game, clone the repository and run the main script from your terminal:

```bash
git clone [[https://github.com/morganeallemann/Snake_Python.git](https://github.com/morganeallemann/Snake_Python.git)]
cd Snake_Python
python3 snake.py
```
## Controls
Arrow keys: Move the snake (Up, Down, Left, Right)
Q / Esc: Quit the game

## Project Structure
snake.py: Entry point and main game loop.
